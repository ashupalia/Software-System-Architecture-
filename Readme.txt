Using Cmd Prompt

1. Open cmd prompt 

2. Change directory to "A20377427-GasPump-CS586" folder using cd cmd
	   
	>cd A20377427-GasPump-CS586

3. Use java command and the jar file "project.jar" to execute

	java -jar project.jar




Using Eclipse

1. Start Eclipse

2. File -> Import -> Existing Projects Into Workspace

4. Browse for the project folder

5. Click on \A20377427-GasPump-CS586\A20377427-GasPump-CS586-Project

6. click "Finish"

7. In the workspace, Right click on the project -> Run -> Run as Java Application
